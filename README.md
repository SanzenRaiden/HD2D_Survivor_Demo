# 项目介绍
本项目是基于 Unity 6 开发的一款 HD‑2D 俯视角生存游戏。
采用 Unity3D 项目搭建地形场景，人物与怪物使用 2D 面片精灵渲染，搭配光影、景深效果，复刻经典 HD‑2D 美术风格。

## 实现功能
1. 角色控制系统：移动、朝向控制，带有受击判定、生命值系统、技能系统、受击特效、伤害显示。
2. 怪物系统：定时随机生成敌人，怪物具备追击仇恨逻辑，受击效果及死亡动画、伤害显示。
3. 场景设置：添加边界空气墙，限制玩家活动范围，地形碰撞完整。
4. 初始UI界面：初始设置按钮UI交互，天赋加点并保留至局内。
5. 局内UI界面：包含血量UI，经验值UI，游戏倒计时UI，死亡提示面板UI，游戏胜利UI。
6. 升级奖励选项UI：升级三选一UI交互对角色进行强化。
7. 代码规范：所有代码添加详细注释。
8. 使用 Git 进行版本管理，配置.gitignore过滤冗余文件。
